安裝Java環境:
1. 到官網下載Java-JDK
2. 解壓到`/usr/local/jvm`目錄
```shell
cd /usr/lib
sudo mkdir jvm
sudo tar -zxvf ./jdk-8u162-linux-x64.tar.gz -C /usr/lib/jvm
```
3. 設置環境變量
```shell
cd ~
vim ~/.bashrc
```
